  - JanusGraph的底层存储使用Hbase
  
图数据库使用场景
- 存量数据大、新增数据多，对实时性要求较高
- 数据倾斜，在不影响吞吐的情况下，降低查询的平均延时；在高可用前提下，对数据一致性、事务性也有要求；
